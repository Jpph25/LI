fact #

Arguments

:: Integer

argumento assume-se não negativo

-> Integer

resultado

A função ’fact’ calcula o factorial (fact n retorna o factorial de um inteiro n).

Alternativamente a função poderia ser definida da seguinte forma:

fact n = if n>0 then n * fact (n-1)
else 1

Exemplos de utilização:

Expand
>>> fact 5
120
>>> fact 0
1

Propriedades:

Expand
fact 0 = 1
n>0 => fact n = n * fact (n-1)

fib :: Integer -> Integer #

A função ’fib’ retorna o n-ésimo elemento da sequência de Fibonacci. Algumas características que distinguem a função ’fib’ da ’fact’:

  • o padrão de recursividade é binário;
  • dispõe de dois casos base.

Exemplos de utilização:

Expand
>>> fib 4
3
>>> fib 10
55
